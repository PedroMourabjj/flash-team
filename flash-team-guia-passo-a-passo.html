5

Criar as tabelas — copie e cole o código abaixo

👉 No menu da esquerda, clique em "SQL Editor"
👉 Clique em "New query"
👉 Apague o que estiver escrito, copie TODO o código abaixo e cole lá:

-- Usuários (alunos e admin) CREATE TABLE usuarios ( id SERIAL PRIMARY KEY, nome TEXT NOT NULL, email TEXT UNIQUE NOT NULL, telefone TEXT, foto_url TEXT, tipo TEXT DEFAULT 'aluno', faixa TEXT, plano TEXT, valor_plano NUMERIC DEFAULT 0, bolsista BOOLEAN DEFAULT false, projeto_social BOOLEAN DEFAULT false, valor_personalizado NUMERIC DEFAULT 0, responsavel_nome TEXT, responsavel_telefone TEXT, created_at TIMESTAMP DEFAULT NOW() ); -- Pagamentos (fonte única de verdade) CREATE TABLE pagamentos ( id SERIAL PRIMARY KEY, user_id INTEGER REFERENCES usuarios(id) ON DELETE CASCADE, mes_referencia TEXT NOT NULL, status TEXT DEFAULT 'pendente', valor NUMERIC DEFAULT 0, data_pagamento DATE, data_vencimento DATE, UNIQUE(user_id, mes_referencia) ); -- Presenças CREATE TABLE presencas ( id SERIAL PRIMARY KEY, user_id INTEGER REFERENCES usuarios(id) ON DELETE CASCADE, data DATE NOT NULL, UNIQUE(user_id, data) ); -- Admin principal INSERT INTO usuarios (nome, email, tipo, faixa) VALUES ('Pedro Moura', 'pedro@flashteam.com', 'admin', 'Preta');

👉 Clique no botão "Run" (ou Ctrl+Enter no teclado)

✅ Se aparecer "Success" em verde — funcionou! Suas tabelas estão criadas.
6

Pegar o endereço de conexão do banco

👉 No menu da esquerda, clique no ícone de engrenagem ⚙️ (Project Settings)
👉 Clique em "Database"
👉 Role até "Connection string"
👉 Selecione a aba "URI"
👉 Copie o texto que começa com postgresql://
👉 Cole num bloco de notas e guarde bem — você vai usar depois

⚠️ Esse texto contém sua senha. Não compartilhe com ninguém.
